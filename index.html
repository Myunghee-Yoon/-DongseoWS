대학생을 위한 AI 역량 강화

응용 생성형 AI 워크숍

단순한 사용자에서 벗어나 AI와 협업하는 'AI 아키텍트'로 진화하세요. 기초 원리부터 실제 앱 개발까지, 당신의 미래를 바꿀 여정이 시작됩니다.

한눈에 보는 AI 모델 지형도

ChatGPT가 시장을 지배하지만, Claude와 Perplexity 같은 전문 모델이 무섭게 성장하고 있습니다. 이 워크숍은 '최고의 AI'가 아닌 '내 과제에 최적의 AI'를 선택하는 전략적 안목을 길러줍니다.

AI와의 소통법, RAFT

효과적인 프롬프트는 AI의 잠재력을 100% 끌어내는 열쇠입니다. RAFT 기법을 통해 AI에게 명확하고 구조적인 지시를 내리는 법을 배워보세요.

👤

Role (역할)

AI에게 '데이터 과학자' 같은 특정 페르소나를 부여하여 답변의 관점과 톤을 설정합니다.

⚙️

Action (행동)

'요약하라', '분류하라' 등 명확한 동사로 수행할 작업을 구체적으로 지시합니다.

📋

Format (형식)

'글머리 기호 목록', 'JSON 객체' 등 원하는 결과물의 구조를 명시적으로 정의합니다.

CONTEXT

Task (과제)

작업에 필요한 배경 정보, 데이터, 제약 조건 등 충분한 맥락을 제공하여 답변의 질을 높입니다.

당신의 개인 연구 조수, NotebookLM

정보의 출처가 불분명한 AI는 이제 그만. 사용자가 제공한 자료에만 근거하여 답변하는 'Grounded AI' NotebookLM으로 학문적 정직성을 지키며 연구 효율을 극대화하세요.

일반 챗봇 (Ungrounded)

  • ✖️출처가 불분명한 인터넷 데이터 기반
  • ✖️'환각 현상'으로 거짓 정보 생성 가능
  • ✖️학술 연구에 사용 시 신뢰도 문제

NotebookLM (Grounded)

  • ✔️100% 사용자가 제공한 자료에만 근거
  • ✔️모든 답변에 명확한 출처(Citation) 제시
  • ✔️신뢰할 수 있는 개인 연구 및 글쓰기 파트너

최종 프로젝트 : '캠퍼스 행사 찾기' 앱 제작 (예시)

워크숍의 모든 것을 종합하여, AI 코드 편집기 Cursor와 함께 실제 웹 애플리케이션을 처음부터 완성하는 전체 개발 사이클을 경험합니다.

1

프로젝트 설정

Vite와 Tailwind CSS로 React 개발 환경을 구축합니다.

2

데이터 구조화

useState를 사용해 행사 정보를 담을 모의 데이터를 생성합니다.

3

컴포넌트 제작

개별 행사를 표시할 'EventCard' 컴포넌트를 만듭니다.

4

UI 렌더링

데이터를 순회하며 EventCard들을 화면에 표시합니다.

5

기능 추가 및 디버깅

등록 폼을 만들고, AI와 함께 오류를 해결하며 앱을 완성합니다.
